# chrome-extension-vue-template
`vue-cli3` + `webpack4` + `element-ui`  实现编译打包Chrome浏览器插件

## 环境安装
```sh
npm install
```

### 开发环境编译并热更新
```sh
npm run serve
or
npm run dev
```

### 生产环境打包
```sh
npm run build
or
npm run build:debug   # 打debug CRX包
```

### 分析包组件大小
```sh
npm run analyze
```

### Lints and fixes files
```sh
npm run lint
```
